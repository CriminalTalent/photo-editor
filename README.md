# 📸 포토 에디터 (Photo Editor)

모던하고 강력한 온라인 포토 에디터입니다.

## ✨ 기능

- 🖼️ 이미지 업로드 및 편집
- 😊 이모지 추가 (모든 기기 이모지 지원)
- ✏️ 텍스트 추가 (색상, 테두리 커스터마이징)
- 🎨 스티커 추가 및 조정
- ✂️ 이미지 자르기 (범위 직접 지정)
- 🖱️ 모든 요소 크기, 회전 조정
- 💾 고해상도 다운로드 (2배 해상도)

## 🚀 빠른 시작

### 로컬에서 실행하기
```bash
npm install
npm run dev
```

### Vercel에 배포하기
1. GitHub에 코드 푸시
2. [vercel.com](https://vercel.com) 접속
3. "Import Project" 클릭
4. 자동 배포 완료!

## 🛠️ 사용 기술
- React 18
- Vite
- Canvas API
